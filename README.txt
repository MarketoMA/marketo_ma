Description
-----------
This module adds Marketo tracking capability to your Drupal site.

Requirements
------------
Drupal 7.x
An active account with Marketo http://www.marketo.com/

Installation
------------
1. Copy the entire marketo directory the Drupal sites/all/modules directory.

2. Login as an administrator. Enable the module in the "Administer" -> "Modules"

3. (Optional) Enable the Marketo User module in the "Administer" -> "Modules"

3. (Optional) Enable the Marketo Webform module in the "Administer" -> "Modules"
